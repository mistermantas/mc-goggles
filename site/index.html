Getting started (Updated)

  1. Download the resource pack, you should get a ZIP file.
  2. Open the folder to which you downloaded the file.
  3. Extract the ZIP file. There should now be a folder called goggles-master.
  4. Right-click that folder and hit Copy.
  5. Now open Minecraft and click Options, then Resource packs.
  6. Next, choose Open resource pack folder.
  7. Paste goggles-master into the resource folder that Minecraft just opened for you.
  8. Go back to Minecraft. Restart the game.
  9. When Minecraft is open again, return to the Options menu. Click Resource packs again. The resource pack should now be listed and you can activate it by choosing it and hitting Done.

For server owners

This pack can also be used on any server (Spigot, Bukkit, Sponge, or some other server software is not required). Pick a version that is suitable for your use case and game version from our releases page, then right-click on the packaged goggles-for-servers.zip file link. Hit ‚ÄúCopy link address.‚Äù Next, you will need to open the server.properties file, and copy that URL to the 35th line.

Save the file and restart the server to apply the changes. Players will now be prompted to use the pack upon logging in. If you see any extra backslashes after restarting the server in your configuration file‚Äîdo not remove them! They are there so that Minecraft can properly interpret the URI.

Your config file should now look something like this:
